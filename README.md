# Progress: BI - API

## Instalação

Depois de clonar o projeto, na raiz do projeto:

```
$ composer install
```

Entrar no arquivo app.php e configurar o acesso ao banco de dados, e então rodar o comando seguinte para criar o banco.

```
$ bin/cake migrations migrate
```
